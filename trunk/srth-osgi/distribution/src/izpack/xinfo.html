You are almost ready to install $APP_NAME - $APP_VER on your machine.

  1. ${application.name} will be installed to the following folder on your machine :
    $app-destination
  2. You will not be able to go back after this step, so check the previous screen for a summary of installation settings.
  3. Please note that the SNOMED CT MySQL Data Pack will require around 1GB space after unpacking!
  4. ${application.name} needs a local installation of MySQL for storing SNOMED CT and its own data.
  5. After installation, please follow the instructions provided in the configuration manual located at : $INSTALL_PATH\docs\${inst.man.name}, to install and configure MySQL for ${application.name} .
  6. The installer will try to open the configuration in your default PDF reader. If it fails, please open it manually and follow the steps described.
  7. An online version of the installation guide is also available.
  8. The post-installation step will verify that ${application.name} can access the local MySQL server using the account that you have configured.

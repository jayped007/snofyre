Read Me for ${application.name} - ${project.version}

Installing ${application.name}

  • This installer will ${application.name} install to a folder of your choice.
  • It is recommended that you run ${application.name} from a folder for which you have read, write and execute permissions.
  • This folder will be referred to as the INSTALLATION-FOLDER in the user manual.

Requirements

  • ${application.name} requires Java Run Time Environment (minimum version 1.5)
  • ${application.name} requires MySQL Server to have been installed on the client machine. You can also configure ${application.name} to connect to a remote MySQL server.
  • The installer will open up the Installation Guide later on to guide you through this process.
